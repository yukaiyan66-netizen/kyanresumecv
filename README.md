# Kyan Yu Resume Website

Static resume website for GitHub Pages.

## Publish with Git

```bash
git init
git add .
git commit -m "Publish resume website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/kyan-resume.git
git push -u origin main
```

Then open the GitHub repository:

Settings → Pages → Build and deployment → Deploy from a branch → `main` → `/(root)` → Save.

The public URL will normally be:

`https://YOUR_USERNAME.github.io/kyan-resume/`
